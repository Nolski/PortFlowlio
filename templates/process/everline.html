Everline

DataVis

Project for HackUpstate utilizing the Evernote API to build Civic Hacking Data Visualizations using Timeline.js

Everline was built to solve the issue of displaying temporal data stored within Evernotes.

Introduction

Upon the discovery of the Timeline.js library built by the Knight Foundation I, along with the rest of Everline's contributors decided that it would be an interesting project to try and take data stored within Evernote and through their API retrieve the data and display it temporally using the Timeline.js library. Everline was awarded the Evernote API prize and an honorable mention at HackUpsate. We are still working with Evernote on making it an official app.

Future Work

Everline currently only works as a method to retrieve temporal data from Evernote which means that the Evernote user must use a specific markup style to determine certain metadata surrounding that event. In the future we hope to use Timeline.js as a method of creating that data as well and storing it in Evernote for future use.
